# ArtEventManager

X（旧Twitter）のタイムラインからAIアートイベント情報を自動的に収集し、管理するためのChrome拡張機能です。

## 概要

この拡張機能をインストールして、対象となるXのポストのURLをコピー＆ペーストするだけで、「AIアートコンテスト」「企画」「締切日」などのイベント情報を抽出します。
収集したデータはサイドパネルで確認でき、カレンダーやガントチャート形式でスケジュールを直感的に管理できます。また、Googleスプレッドシートとの連携によるバックアップも可能です。

## 主な機能

- **イベント情報抽出**: 特定のポストURLを入力するだけで、イベントの概要や締切日を簡単に抽出。
- **カレンダー表示**: イベントの締切日を月間カレンダーで確認。
- **ガントチャート**: 複数のイベントの開催期間をタイムラインで把握。
- **参加進捗管理**: 「未着手」「完了」のステータスを手動で管理。
- **Googleスプレッドシート連携**: 収集したデータを自動的にGoogleスプレッドシートへ保存・同期。

## インストール方法

1. このリポジトリをダウンロード（クローン）します。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をONにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードした `ArtEventManager` フォルダを選択します。

## セットアップ (Google Sheets連携)

Googleスプレッドシートと連携させるには、以下の手順でクライアントIDの設定が必要です。

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクトを作成します。
2. 「Google Sheets API」を有効にします。
3. 「OAuth 同意画面」を設定します。
4. 「認証情報」から「OAuth 2.0 クライアント ID」を作成します（アプリケーションの種類は「Chrome アプリ」）。
    - アプリケーション ID に、Chromeの拡張機能管理画面で表示されているIDを入力してください。
5. 発行されたクライアントIDを `manifest.json` の `oauth2.client_id` フィールドに記入します。
